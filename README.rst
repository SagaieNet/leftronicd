##########
Leftronicd
##########

A script to periodically post information to leftronic.com


*************
Configuration
*************

Configuration is done in yaml.

accesskey
=========

Your leftronic API access key.



*******
Running
*******

``leftronicd <configfile> [-v]``


Example
=======

See mssql_sample.yml file for a sample configuration file used to connect to a MS-SQL database (SQL Server)
